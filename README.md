🔹 App Overview 🔹
 
This app tracks Dota 2 in-game time and provides stack, rune, wisdom, and tormentor reminders using Game State Integration and reminds you while ingame.


  
🔹 Features 🔹
 
Stack reminders (xx:43 prepare, xx:45 stack)
Wisdom rune alerts
Tormentor preparation alerts
Pause-safe timing (GSI-based)
Lightweight overlay / no HUD version

🔹 Installation Guide 🔹

  
1. Download the .exe from Releases
2. Add Game State Integration file to:
   Steam\steamapps\common\dota 2 beta\game\dota\cfg\gamestate_integration\
3. Add launch option:
   -gamestateintegration
4. Run the app




This app:

Does NOT modify Dota 2 files
Does NOT interact with memory
Only reads official Game State Integration data
Is fully open source for verification
